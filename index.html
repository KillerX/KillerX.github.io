Curl Data From File

CURL - Fill form field from file In case you need to put a fairly large amount of data into a form field using CURL, the normal procedures do not seem to work (at least if GO is on the receiving side). Attempting to do curl -X POST -F field=@filename.txt runs and the data is technically sent but it is sent like this: 00ed: --------------------------6742981c1eec2761 0119: Content-Disposition: form-data; name="field"; filename="filename.
